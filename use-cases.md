# .pyLee Use Cases

## Example Requests

### Data Cleaning
- Clean a messy CSV file
- Standardize column names
- Remove duplicates
- Fill or flag missing values
- Export cleaned data to Excel

### Excel Automation
- Merge multiple Excel files
- Create summary sheets
- Generate formatted reports
- Convert CSV to Excel automatically

### Web Scraping
- Scrape product listings
- Extract table data from websites
- Monitor websites for updates
- Save scraped data into CSV or Excel

### PDF Work
- Extract tables from PDFs
- Pull text from reports
- Convert extracted information into structured datasets

### Monitoring Bots
- Check a website every 10 minutes
- Detect content changes
- Log updates automatically

### ETL / Engineering Tasks
- Convert raw logs into structured data
- Transform JSON into tabular format
- Validate input files before processing
- Build quick automation scripts for repetitive tasks
